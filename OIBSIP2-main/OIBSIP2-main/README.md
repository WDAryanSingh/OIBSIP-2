# Tribute Page – Lionel Messi

## Objective

The objective of this project is to create a tribute page for a person I admire
using basic HTML and CSS. Lionel Messi was chosen for this project because of
his extraordinary football skills, humility, discipline, and inspirational
journey.

This project focuses on practicing layout design, typography, styling, and
content presentation while creating a visually appealing web page.

---

## Tools & Technologies Used

- **HTML** – To structure the tribute page content  
- **CSS** – To design the layout, colors, animations, and responsiveness  
- **VS Code** – Code editor used for development  
- **Web Browser** – For testing and previewing the page  

---

## Steps Performed

1. Created the basic structure of the page using HTML  
2. Added sections describing Lionel Messi’s early life, career, and legacy  
3. Included an image to enhance visual appeal  
4. Designed a modern dark-themed layout with gold and blue highlights  
5. Applied CSS animations, hover effects, and transitions  
6. Ensured the page is responsive across different screen sizes  

---

## Features

- Premium and modern UI design  
- Attractive color scheme inspired by football and greatness  
- Clean typography and structured sections  
- Smooth hover and animation effects  
- Fully responsive layout  

---

## Outcome

The final result is a clean, eye-catching, and informative tribute page dedicated
to Lionel Messi. This project helped improve my understanding of HTML structure,
CSS styling, layouts, animations, and responsive design principles.
